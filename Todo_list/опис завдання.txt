Мета
Реалізувати веб-додаток для управління завданнями, який включає такі функції:

Додавання завдань.
Перегляд списку всіх завдань.
Редагування завдань.
Видалення завдань.
Етапи виконання
Створення бази даних:

Таблиця tasks з полями:
id (цілий, автоінкремент, первинний ключ).
description (текст, не порожнє поле).
created_at (дата й час створення).
updated_at (дата й час останнього оновлення).
Структура проєкту:

Папка App: місце для класів і моделей.
Папка public: для файлів, доступних через веб-сервер (HTML, CSS, JS).
Файл Task.php: модель для роботи з базою даних.
Файл index.php: головна сторінка програми.
Файл edit_task.php: сторінка редагування завдань.
Файл delete_task.php: обробник видалення завдань.
Функціонал програми:

Додавання завдання:
Форма для введення опису завдання.
Перевірка введених даних (валідація).
Запис до бази даних через модель Task.
Перегляд завдань:
Отримання списку завдань із бази даних.
Відображення в HTML.
Редагування завдання:
Завантаження існуючого завдання для редагування.
Оновлення опису завдання в базі даних.
Видалення завдання:
Видалення завдання за його ідентифікатором.
Додаткові вимоги:

Захист від SQL-ін'єкцій (використання PDO).
Валідація даних.
Очищення вхідних даних (наприклад, htmlspecialchars()).
